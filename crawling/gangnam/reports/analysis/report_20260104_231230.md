# 강남언니 병원 데이터 수집 리포트

## 1. 수집 개요
- **수집 대상**: 강남언니 병원 정보 (https://www.gangnamunni.com/hospitals)
- **수집 항목**: 병원 이름, 별 평점, 평가(리뷰) 개수, 이벤트 개수
- **수집 일시**: 2026년 1월 4일

## 2. 수집 방법 및 도구
- **수집 방법**: 강남언니 내부 API (`/api/solar/search/hospitals`)를 통한 데이터 요청 및 파싱
- **사용 도구**:
  - **Python**: 데이터 수집 및 처리 메인 언어
  - **Requests**: HTTP 통신을 위한 라이브러리 (API 호출 시 사용)
  - **Pandas**: 수집된 데이터를 구조화하고 CSV 파일로 저장하기 위한 라이브러리
  - **Datetime**: 파일명에 타임스탬프를 부여하기 위해 사용

## 3. 수집 과정
1. 강남언니 웹사이트의 네트워크 트래픽 분석을 통해 병원 목록을 가져오는 API 엔드포인트와 필요한 헤더(Cookie, Token 등)를 확인하였습니다.
2. `districtCodeList`를 포함한 모든 요청 파라미터를 설정하여 API를 호출하였습니다.
3. API 응답(JSON)에서 필요한 정보(`name`, `rating`, `ratingCount`, `eventCount`)를 추출하였습니다.
4. `start` 파라미터를 조정하며 페이지네이션을 처리하였습니다. (현재 테스트를 위해 100개 데이터 수집으로 제한 설정)
5. 수집된 데이터를 Pandas DataFrame으로 변환 후 CSV 형식으로 저장하였습니다.
