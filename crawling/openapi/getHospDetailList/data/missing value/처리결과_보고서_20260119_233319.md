### 대체값 해석
1. **0 값**: "없음" 또는 "0개"를 의미 (정보 미제공 아님)
2. **'N' 값**: "해당 없음" 또는 "운영 안함"을 의미
3. **'정보없음'**: 연락처 정보가 제공되지 않음
4. **NULL**: 정보 미제공 또는 해당 사항 없음

1. **통계 분석**: 0 대체된 값은 실제 0과 구분 필요 시 별도 플래그 활용
