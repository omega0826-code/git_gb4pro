## 🤔 어떻게 사용하나요?

hwp-mcp는 **한글 프로그램을 AI가 원격으로 조종**하는 방식으로 동작합니다.

### 동작 원리

```
나(사람) → AI에게 요청 → AI가 MCP 도구 호출 → 한글 프로그램이 실제로 동작 → 결과 반환
```

예시:
1. 나: *"새 한글 문서를 만들고 제목을 써줘"*
2. AI가 내부적으로 `hwp_create`, `hwp_insert_text` 도구를 호출
3. **한글 프로그램이 실제로 열리고**, 문서가 생성되며, 텍스트가 입력됩니다

> 💡 **핵심**: hwp-mcp를 사용하면 **한글 프로그램 화면이 실제로 움직이는 것**을 볼 수 있습니다.  
> 사람이 한글을 사용하는 것처럼, AI가 대신 한글 프로그램을 조작합니다.

### 사용 전 주의사항

1. **한글 프로그램이 실행 가능한 상태**여야 합니다
2. **다른 한글 문서를 편집 중이면 충돌**할 수 있습니다
3. AI가 작업하는 동안 **한글 프로그램을 직접 조작하지 마세요**

## 📝 기본 사용법: 문서 만들기

### 새 문서 만들기

```
요청: "새 한글 문서를 만들어줘"
```

AI가 `hwp_create` 도구를 호출 → 한글 프로그램이 열리고 빈 문서가 생성됩니다.

### 텍스트 입력하기

```
요청: "문서에 '안녕하세요, 이것은 테스트 문서입니다.'라고 입력해줘"
```

AI가 `hwp_insert_text` 도구를 호출 → 현재 커서 위치에 텍스트가 입력됩니다.

### 문서 저장하기

```
요청: "현재 문서를 D:\문서\테스트.hwp로 저장해줘"
```

AI가 `hwp_save` 도구를 호출 → 지정한 경로에 .hwp 파일로 저장됩니다.

### 기존 문서 열기

```
요청: "D:\문서\보고서.hwp 파일을 열어줘"
```

AI가 `hwp_open` 도구를 호출 → 한글 프로그램에서 해당 파일이 열립니다.

## 🔤 글꼴 설정

### 글꼴 크기 변경

```
요청: "글꼴 크기를 20pt로 설정해줘"
```

### 굵은 글씨 설정

```
요청: "글꼴을 굵게 설정해줘"
```

### 글꼴 종합 설정

```
요청: "글꼴을 '나눔고딕', 크기 16pt, 굵게로 설정해줘"
```

> 💡 글꼴 설정은 **이후에 입력되는 텍스트**에 적용됩니다.  
> 이미 입력된 텍스트의 글꼴을 바꾸려면 먼저 해당 텍스트를 선택해야 합니다.

## 📊 표(테이블) 작업

### 표 만들기

```
요청: "5행 3열 표를 만들어줘"
```

AI가 `hwp_insert_table` 도구를 호출 → 현재 커서 위치에 표가 생성됩니다.

### 표에 데이터 채우기

```
요청: "표에 다음 데이터를 채워줘:
이름, 나이, 부서
김철수, 30, 개발팀
이영희, 28, 기획팀
박민수, 35, 영업팀
홍길동, 42, 인사팀"
```

AI가 `hwp_fill_table_with_data` 도구를 호출 → 각 셀에 데이터가 입력됩니다.

**`has_header` 옵션:**
- `true`로 설정하면 첫 번째 행이 **헤더(제목 행)** 로 처리됩니다
- 헤더 행은 굵은 글씨 등 별도 서식이 적용될 수 있습니다

### 열에 연속 숫자 채우기

```
요청: "표의 첫 번째 열에 1부터 10까지 번호를 넣어줘"
```

AI가 `hwp_fill_column_numbers` 도구를 호출 → 지정한 열에 순서대로 번호가 입력됩니다.

### 표 관련 팁

- **표 안에 표 생성 방지**: 커서가 이미 표 안에 있으면 표가 중첩됩니다. AI에게 "표 바깥에서" 라고 명시하세요.
- **데이터가 많을 때**: 직접 데이터를 나열하는 것보다 파일에서 읽어오는 방식이 효율적입니다.

## ⚡ 일괄 작업 (배치)

여러 작업을 한꺼번에 실행할 수 있습니다. 하나씩 요청하는 것보다 빠르고 효율적입니다.

### 기본 사용법

```
요청: "다음 작업을 순서대로 한꺼번에 실행해줘:
1. 새 문서 만들기
2. '월간 업무 보고서'를 제목으로 입력
3. 글꼴 크기 24pt, 굵게 설정
4. 아래에 '2026년 2월' 입력
5. D:\문서\월간보고서_202602.hwp로 저장"
```

AI가 `hwp_batch_operations` 도구를 호출 → 모든 작업이 순서대로 실행됩니다.

### 배치 작업의 장점

| 방식                | 소요 시간               | 에러 처리                |
| ------------------- | ----------------------- | ------------------------ |
| 한 번에 하나씩 요청 | 느림 (각 요청마다 대기) | 중간에 실패 시 수동 복구 |
| 배치 작업           | 빠름 (한번에 전달)      | 에러 발생 위치 자동 보고 |

### 알아두면 좋은 것

- **AI가 한글을 못 찾을 때**: 한글 프로그램을 한 번 수동으로 실행했다가 닫은 후 다시 시도하세요
- **보안 경고가 뜰 때**: "허용"을 눌러주세요. 또는 설치 가이드의 보안 모듈을 등록하세요
- **문서가 깨질 때**: `git pull`로 서버를 최신 버전으로 업데이트하세요

## 🆘 문제가 생겼을 때

### 자주 묻는 질문 (FAQ)

**Q: 한글 프로그램이 열리지 않아요**  
A: 한글 프로그램을 수동으로 한 번 실행해 보세요. 정상 실행되면 닫고 다시 시도합니다.

**Q: "COM 오류"가 발생해요**  
A: 한글 프로그램이 이미 다른 작업 중이거나, 관리자 권한이 필요할 수 있습니다.  
PowerShell을 관리자 권한으로 실행해 보세요.

**Q: 표에 데이터가 잘못 들어가요**  
A: `git pull`로 최신 버전을 받으세요. 표 관련 버그가 2025-03-27에 수정되었습니다.

**Q: macOS에서 사용하고 싶어요**  
A: 이 서버는 Windows 전용입니다. macOS에서는 `hwpx-mcp-server`를 사용하세요.
